Connect Firebase to Go Live

Green Light is currently saving posts locally. Follow these steps to share with the team:

  1. Go to console.firebase.google.com → create a free project (skip Google Analytics).
  2. Click Build → Firestore Database → Create database. Choose Start in test mode, pick any region.
  3. Click the gear → Project settings, scroll to "Your apps", click </> (web), register a name, copy the firebaseConfig object.
  4. Open this HTML file, find FIREBASE_CONFIG near the top of the <script> block, paste your values.
  5. Save & refresh — the banner turns green and posts sync in real time.

Firestore's free tier handles ~50k reads/day. For production, update security rules to restrict writes.
